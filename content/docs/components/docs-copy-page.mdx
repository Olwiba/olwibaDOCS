## Installation

```sh
npm install @olwiba/docs
```

## Usage

```tsx
import { DocsCopyPage } from '@olwiba/docs';
```

```tsx
<DocsCopyPage
  page={rawContent}
  url={typeof window !== 'undefined' ? window.location.href : '/docs'}
/>
```

`rawContent` is typically obtained from the server loader via `page.data.getText('raw')`.
