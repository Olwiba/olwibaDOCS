## Installation

```sh
npm install @olwiba/docs
```

## Usage

```tsx
import { ThemeCodeBlock } from '@olwiba/docs';
```

```mdx
<ThemeCodeBlock />
```

Registered in `mdxComponents` so it works directly in MDX content. Requires an `ActiveThemeProvider` ancestor.
