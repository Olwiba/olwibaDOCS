## Import

```tsx
import { CodeFence } from '@olwiba/docs';
```

## Usage

### As a direct component

```tsx
<CodeFence code="const hello = 'world';" />
```

### As an MDX pre replacement

`CodeFence` is automatically used as the `pre` element in the `mdxComponents` mapping, so all fenced code blocks in your MDX content use it:

```tsx
import { mdxComponents } from '@olwiba/docs';

// mdxComponents.pre renders CodeFence
<MDX components={{ ...defaultMdxComponents, ...mdxComponents }} />
```
