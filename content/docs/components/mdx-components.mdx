## Import

```tsx
import { mdxComponents } from '@olwiba/docs';
// or
import { mdxComponents } from '@olwiba/docs/mdx';
```

## Usage

Use the components with fumadocs-ui's default MDX components:

```tsx
import defaultMdxComponents from 'fumadocs-ui/mdx';
import { mdxComponents } from '@olwiba/docs';

<MDX
  components={{
    ...defaultMdxComponents,
    ...mdxComponents,
  }}
/>
```

## Included Components

### Typography

- `h1`, `h2`, `h3`, `h4` — Styled headings with proper spacing and scroll margins
- `p` — Paragraphs with comfortable line height
- `a` — Links with underline styling
- `strong` — Bold text
- `blockquote` — Styled block quotes

### Lists

- `ul` — Unordered lists
- `ol` — Ordered lists
- `li` — List items

### Code

- `pre` — Renders via [CodeFence](/docs/components/code-fence) with copy button and overflow fade
- `code` — Inline code with muted background

### Tables

- `table`, `tr`, `th`, `td` — Styled tables with borders and scroll wrapper

### Special Components

- `Steps` / `Step` — Numbered step-by-step instructions
- `Tabs`, `TabsList`, `TabsTrigger`, `TabsContent` — Tabbed content
- `Callout` — Highlighted information boxes
- `Button` — Interactive buttons
- `Accordion` — Collapsible content sections
- `Alert` — Alert messages
- [CodeFence](/docs/components/code-fence) — Enhanced code blocks (also used as `pre`)
- [APIReference](/docs/components/api-reference) — Collapsible prop tables
- [ThemeSelector](/docs/components/theme-selector) — Theme switcher dropdown
- [ThemeCodeBlock](/docs/components/theme-code-block) — Theme CSS copier
- `Link` — TanStack Router Link with underline styling
