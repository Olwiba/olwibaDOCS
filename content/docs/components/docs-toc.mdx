## Import

```tsx
import { DocsToc, type TocItem } from '@olwiba/docs';
```

## Usage

```tsx
import { DocsToc } from '@olwiba/docs';

function DocsPage({ toc }) {
  return (
    <div className="sticky top-16">
      <DocsToc toc={toc} />
    </div>
  );
}
```

### Extracting TOC from MDX

The TOC is typically extracted during server-side loading:

```tsx
const serverLoader = createServerFn()
  .handler(async ({ data: slugs }) => {
    const page = source.getPage(slugs);

    return {
      toc: (page.data.toc ?? []).map((item) => ({
        title: extractTextFromReactNode(item.title),
        url: item.url,
        depth: item.depth,
      })),
    };
  });
```
