## Installation

```sh
npm install @olwiba/docs
```

## Usage

```tsx
import { ThemeSelector } from '@olwiba/docs';
```

```tsx
<ThemeSelector />
```

Place it anywhere inside an `ActiveThemeProvider`. Also registered in `mdxComponents` for direct MDX use:

```mdx
<ThemeSelector />
```
