## Import

```tsx
import { DocsMobileNav } from '@olwiba/docs';
```

## Usage

### 1. Add the trigger container in your header

```tsx title="SiteHeader.tsx"
<div id="docs-mobile-nav-trigger" className="empty:hidden lg:hidden" />
```

### 2. Add DocsMobileNav in your docs route

```tsx title="routes/docs/$.tsx"
<DocsMobileNav tree={data.pageTree} sections={sidebarSections} />
```

The component automatically portals the trigger button into the `#docs-mobile-nav-trigger` element.
