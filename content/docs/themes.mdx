## Available Themes

The following themes are available:

| Theme | Description |
|-------|-------------|
| Default | Neutral grayscale theme |
| Emerald | Fresh green tones |
| Blue | Classic blue accent |
| Purple | Vibrant purple highlights |
| Rose | Warm rose/pink tones |
| Orange | Energetic orange accent |
| Slate | Professional slate blue |

## Installation

To use themes in your project, you need to:

1. Copy the theme CSS variables to your global stylesheet
2. Wrap your app with the `ActiveThemeProvider`
3. Use the `ThemeSelector` component or create your own

### Step 1: Add Theme CSS

Copy the CSS variables for your chosen theme and add them to your global CSS file.

<ThemeCodeBlock />

### Step 2: Add the Theme Provider

Wrap your application with the `ActiveThemeProvider`:

```tsx
import { ActiveThemeProvider } from '@olwiba/docs';

export function App({ children }) {
  return (
    <ActiveThemeProvider>
      {children}
    </ActiveThemeProvider>
  );
}
```

### Step 3: Add Theme Selector (Optional)

Add the theme selector to allow users to switch themes:

```tsx
import { ThemeSelector } from '@olwiba/docs';

export function Settings() {
  return (
    <div>
      <h2>Theme</h2>
      <ThemeSelector />
    </div>
  );
}
```

## Custom Themes

You can create your own themes by defining CSS variables in the same format. The key variables are:

| Variable | Description |
|----------|-------------|
| `--primary` | Main accent color |
| `--primary-foreground` | Text on primary backgrounds |
| `--ring` | Focus ring color (usually matches primary) |
| `--background` | Page background |
| `--foreground` | Default text color |
| `--muted` | Muted background |
| `--muted-foreground` | Muted text |
| `--accent` | Accent background (hover states) |
| `--border` | Border color |
| `--surface` | Surface background (cards, code blocks) |
| `--code` | Code block background |
| `--code-foreground` | Code block text |
| `--code-highlight` | Highlighted code line background |
| `--blueprint-fg` | Blueprint pattern color |

All colors use the `oklch()` color space for perceptual uniformity.
